# how-many-deps

Count how many deps are in your repo

```bash
$ npx how-many-deps

  You have:
    54 dependencies!
      (both direct and indirect across 1 repo)

```

and in a monorepo

```bash
$ npx how-many-deps

  You have:
    2175 dependencies!
      (both direct and indirect across 28 repos)

```
